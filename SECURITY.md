# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any plugin or configuration in this repository, please report it responsibly.

**Email**: Open a private security advisory via GitHub's [security tab](../../security/advisories/new).

Please include:
- Description of the vulnerability
- Steps to reproduce
- Affected plugin(s)

We'll acknowledge your report within 48 hours and work on a fix.

## Scope

This repository contains plugin configurations and documentation. Security concerns typically involve:
- MCP server configurations pointing to untrusted endpoints
- Plugins requesting excessive permissions
- Credential or secret exposure in plugin configs
